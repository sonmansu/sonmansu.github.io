# 가상 데스크탑 단축키
- 모든 가상 데스크톱보기 : Windows Key + Tab
- 새 가상 데스크톱 만들기 : Windows Key + Ctrl + D
- 현재 가상 데스크톱 닫기 : Windows Key + Ctrl + F4
- 다음 가상 데스크톱으로 이동 : Windows Key + Ctrl + Right
- 이전 가상 데스크톱으로 이동 : Windows Key + Ctrl + Left
